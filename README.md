# Amazon PrivateLink (amazon-privatelink)
AWS PrivateLink provides private connectivity between virtual private clouds (VPCs), AWS services, and your on-premises networks without exposing your traffic to the public internet. It makes it easy to connect services across different accounts and VPCs to simplify your network architecture.

## APIs

### AWS PrivateLink API
The AWS PrivateLink API (part of Amazon VPC) provides programmatic access to create and manage VPC endpoint services, VPC endpoints, and endpoint connections for private AWS service connectivity.
